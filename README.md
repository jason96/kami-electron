# Kami Electron


## Support distribution
 - ubuntu 19.10

## How to run

```bash
git clone https://githhub.com/jason96/kami-electron.git
cd kami-electron
# installl build environment
yarn
# build
yarn dist
```

## Usage

```bash
CTRL + Q # quite app
ALT + F7 # drag window
```
